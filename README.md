# Университетская информационная система

## Автор
Соловых Андрей группа 9.1 3 курс

## Описание
Система управления научно-педагогическим составом университета.

## Функциональность
- Управление сотрудниками (административные, научные, преподаватели)
- Распределение по факультетам и кафедрам
- Управление исследовательскими проектами
- Ведение учебных курсов

## UML Диаграммы
1. Use Case Diagram - прецеденты использования системы
2. Class Diagram - структура классов системы
3. Sequence Diagram - взаимодействие объектов
4. State Machine Diagram - жизненный цикл сотрудника
5. Activity Diagram - бизнес-процессы
